MemeCoin Whitepaper

A Decentralized Staking and Reward System on Shibarium with WoofSwap Integration

1. Introduction

MemeCoin is a decentralized token system built on the Shibarium blockchain, designed to empower communities to create meme-themed cryptocurrencies. Integrated exclusively with WoofSwap liquidity pools, MemeCoin enables users to stake WoofSwap LP tokens or the MemeCoin itself to earn rewards. Leveraging Shibarium's efficiency and WoofSwap's liquidity, it offers a fun and rewarding experience for global participants.

This whitepaper details the system's mechanics, deployment process, participation rules, and reward structure.

2. System Overview

2.1 Core Components

  • MemeCoin Contract: Manages token creation, staking, and rewards.
  • MemeCoinFactory Contract: Deploys new MemeCoin instances.
  • Staking DApp: A user interface for deployment and interaction.

2.2 Key Features

  • Customizable token name and symbol.
  • Staking of WoofSwap LP tokens and MemeCoins.
  • Dynamic reward decay.
  • Fee collection for creators via WoofSwap pairs.
  • Supply cap with mining termination.

3. Tokenomics

3.1 Token Specifications

  • Decimals: 18
  • Maximum Supply: 1,000,000,000,000,000 tokens (1 quadrillion)
  • Stake Units:
    • WoofSwap LP: Configurable per deployment (e.g., 1,000,000 wei)
    • MemeCoin: Fixed at 10,000 tokens

3.2 Reward Structure

Rewards decrease over time based on staking activity:

  • Initial LP Reward: 500 tokens/block/unit
  • Minimum LP Reward: 10 tokens/block/unit
  • Initial MemeCoin Reward: 25,000 tokens/block/unit
  • Minimum MemeCoin Reward: 500 tokens/block/unit
  • Reward Decay: 2% reduction every 600,000 blocks (~1 month)
  • Maximum Reward Blocks: 29,400,000 (~5 years)
  • Max Reward per Claim: 1,000,000 tokens

Mining ends when the total supply hits the cap.

4. Deployment Process

Deploy a MemeCoin using the MemeCoinFactory via the DApp, requiring a WoofSwap LP token.

4.1 Requirements

  • Wallet connected to Shibarium (e.g., MetaMask).
  • BONE tokens for gas fees.
  • WoofSwap LP token address (from WoofSwap on Shibarium).

4.2 Steps

  1. Connect wallet to the DApp on Shibarium (Chain ID: 109).
  2. Go to "Deploy MemeCoin."
  3. Input:
    • Name: e.g., "DogCoin"
    • Symbol: e.g., "DOG"
    • LP Token Address: WoofSwap LP token address
    • Stake Unit LP: Minimum LP staking unit (e.g., 1,000,000 wei)
  4. Click "Deploy MemeCoin" and confirm.
  5. View the contract address in "Deployed MemeCoins."

The deployer (owner) collects fees from the WoofSwap LP pair.

5. Participation Rules

Interact with deployed MemeCoins via the DApp.

5.1 Staking WoofSwap LP Tokens

  • Requirement: WoofSwap LP tokens.
  • Amount: Multiple of the Stake Unit LP.
  • Steps:
    1. Search MemeCoin address in DApp.
    2. Enter LP amount.
    3. Approve contract to spend LP tokens.
    4. Confirm staking.

5.2 Staking MemeCoins

  • Requirement: Own the MemeCoin.
  • Amount: Multiple of 10,000 tokens.
  • Steps:
    1. Search MemeCoin address.
    2. Enter MemeCoin amount.
    3. Confirm staking.

5.3 Claiming Rewards

Click "Claim Rewards" to mint rewards to your balance.

5.4 Unstaking

  • LP: "Unstake All" withdraws LP tokens and claims rewards.
  • MemeCoin: "Unstake All" withdraws MemeCoins and claims rewards.

5.5 Fee Collection (Owner Only)

Fees from WoofSwap LP pairs are collected by the owner during staking/unstaking.

6. Security Features

  • Non-Reentrant: Prevents reentrancy attacks.
  • SafeERC20: Ensures safe transfers.
  • Ownership: Fees restricted to deployer.
  • Supply Cap: Mining stops at 1 quadrillion.
